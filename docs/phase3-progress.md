## 功能特点

### 闹钟
- 直观的时间选择（滚轮）
- 灵活的重复规则（每天/工作日/周末/自定义）
- 智能重复文本显示
- 贪睡功能支持
- 本地持久化存储

### 计时器
- 高精度计时（10ms）
- 无限计次支持
- 单次时间和累计时间显示
- 暂停后可继续
- 清晰的计次列表

### 倒计时
- 快速设置预设时间
- 圆形进度可视化
- 倒计时完成提醒
- 暂停/继续功能
- 重置到初始时间

## 技术实现

### 定时器精度
- 闹钟：秒级（1秒）
- 计时器：10毫秒
- 倒计时：100毫秒

### 状态管理
- GetX 响应式状态
- Obx 自动更新 UI
- 控制器生命周期管理

### 数据持久化
- JSON 序列化/反序列化
- SharedPreferences 存储
- 异步加载/保存
