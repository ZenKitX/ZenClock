# ZenClock 设计规范

## 禅意设计理念

ZenClock 的设计灵感来源于禅宗美学，追求：
- **极简主义**: 去除一切不必要的元素
- **自然和谐**: 使用自然界的色彩和质感
- **专注当下**: 让用户专注于时间本身

## 配色方案

## 字体规范

### 时间显示
- **字号**: 80sp（时分）、32sp（秒）
- **字重**: Light (300)
- **字体**: 系统默认（保持简洁）

### 辅助文字
- **字号**: 14-16sp
- **字重**: Regular (400)
- **透明度**: 60-80%

## 动画规范

### 时序
- **快速响应**: 150ms（按钮按压）
- **柔和过渡**: 200-300ms（状态切换）
- **缓慢呼吸**: 800ms（主题切换）

### 曲线
- **标准**: `easeInOut`
- **弹性**: `easeInOutCubic`
- **自然**: `decelerate`

## Neumorphic 设计

### 浅色主题
```dart
阴影（凸起）:
  - 深色阴影: Color(0x40000000), offset: (4, 4), blur: 8
  - 浅色高光: Color(0xFFFFFFFF), offset: (-4, -4), blur: 8

阴影（凹陷）:
  - 深色阴影: Color(0x40000000), offset: (-2, -2), blur: 4
  - 浅色高光: Color(0xFFFFFFFF), offset: (2, 2), blur: 4
```

### 深色主题
```dart
阴影（凸起）:
  - 深色阴影: Color(0x60000000), offset: (4, 4), blur: 8
  - 浅色高光: Color(0x20FFFFFF), offset: (-4, -4), blur: 8
```

## 布局规范

### 间距
- **小间距**: 8dp
- **中间距**: 16dp
- **大间距**: 24dp
- **超大间距**: 32dp

### 圆角
- **小圆角**: 8dp（按钮）
- **中圆角**: 16dp（卡片）
- **大圆角**: 24dp（容器）

## 交互规范

### 触觉反馈
| 操作 | 震动时长 | 强度 |
|------|---------|------|
| 轻触 | 10ms | 50 |
| 按压 | 15ms | 80 |
| 确认 | 20ms | 120 |

### 音效
- **音量**: 50-70%
- **类型**: 自然音效（竹子、水滴、风铃）
- **时长**: 200-500ms

## 可访问性

- **对比度**: 符合 WCAG AA 标准（4.5:1）
- **触摸目标**: 最小 48x48dp
- **语义化**: 使用 Semantics Widget
- **屏幕阅读器**: 支持 TalkBack/VoiceOver
